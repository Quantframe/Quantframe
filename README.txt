V.2018.11.10
这份代码可以有效帮助想跟踪实盘交易的Quant们每天自动更新当日的数据
当前已有下载股票和指数，后续会加入下载tushare上有的其他品种，并加入每日实时数据自动刷新，并在收盘自动存入历史数据库的功能

1，数据储存格式：可自行在KDB数据库文件,和CSV中选择
2，通过在config_download.json中调整参数和存贮地址等，通过运行Data_Management/download/HDB.py 来运行程序
3，本程序基于tushare
4，程序尚在完善过程中，目前只完善了股票数据每日追溯更新功能
